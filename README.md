# ILE Config Excel

## Excel Structure

### Tabs

Both ILE and VE Excel Config Processors are divided into tabs, each of a specigic type as identified by its highlighted colour (see table).

| **Type** | **Colour** | Function                                                                                               |
| -------------- | ---------------- | ------------------------------------------------------------------------------------------------------ |
| Input          | Green            | User is expected to paste data into sheet.                                                             |
| Output         | Blue             | Generated data is exported to sheet                                                                    |
| Processing     | Red              | Data is parsed through sheet, either containing formulae or blank and used for temporary data storage. |
| Mapping        | Grey             | Sheet contains a table mapping sets of values together,Â used in lookups.                              |
